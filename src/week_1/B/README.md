# Problem B. Multiplication 1

**Time limit:** 2000 ms  
**Memory limit:** 1048576 kB  

### Problem Statement
Compute \( A **x** B \).

### Constraints
- All values in input are integers.
- \( 1 <= A <= 100 \)
- \( 1 <= B <= 100 \)

### Input
Input is given from Standard Input in the following format:
```
A B
```

### Output
Print the value as an integer.

### Sample 1
**Input:**
```
2 5
```
**Output:**
```
10
```


### Sample 2
**Input:**
```
100 100
```
**Output:**
```
10000
```
